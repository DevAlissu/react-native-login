# Projeto Portfólio Login - Aplicação Mobile React Native

</div>

<br>

<div align="center">
  
[![Generic badge](https://img.shields.io/badge/Made%20by-Alissu-purple.svg)](https://shields.io/) [![made-for-VSCode](https://img.shields.io/badge/Made%20for-VSCode-1f425f.svg)](https://code.visualstudio.com/) [![npm version](https://badge.fury.io/js/react-native.svg)](https://badge.fury.io/js/react-native)

Aplicação Front-end Mobile desenvolvida em React Native para layout de Tela de Login para aplicativos diversos e adaptação de projetos anteriores, com animação no formulário no estilo estilingue, além do efeito de diminuir e aumentar o logotipo de acordo com o teclado aberto.

<div align="center">

![login](https://user-images.githubusercontent.com/48495838/84698552-5a7d4a80-af26-11ea-98ae-9c8074e8e53e.png)

</div>

## :rocket: Tecnologias
<ul>
  <li>Expo</li>
  <li>StatusBar</li>
  <li>KeyboardAvoidingView</li>
  <li>TextInput</li>
  <li>TouchableOpacity</li>
  <li>StyleSheet</li>
  <li>Animated</li>
  <li>Keyboard</li>
  <li>useState</li>
  <li>useEffect</li>
</ul>

## :arrow_forward: Start
<ul>
  <li>npm install</li>
  <li>npm run start / npm start</li>
</ul>
